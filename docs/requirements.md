# VISAVA

## Theme
AI-Powered Mental Wellness and Early Intervention Platform

## Aim
To monitor mental well-being and provide personalized interventions.

## Version 1 Features
1. Register/Login
2. PHQ-9 Assessment
3. GAD-7 Assessment
4. Daily Tracking
   - Mood
   - Stress
   - Sleep
   - Energy
   - Productivity
5. Journal Entries
6. Dashboard
7. Basic Recommendations

## Database Tables
- users
- assessments
- daily_logs
- journal_entries
- recommendations
